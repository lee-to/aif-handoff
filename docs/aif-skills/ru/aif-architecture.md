# /aif-architecture — Архитектурные гайдлайны

Анализирует tech stack проекта, рекомендует архитектурный паттерн и создаёт `ARCHITECTURE.md`.

## Использование

```
/aif-architecture
```

## Что создаёт

Файл `.ai-factory/ARCHITECTURE.md` с:

- Выбранным архитектурным паттерном (layered / hexagonal / microservices / etc.)
- Структурой папок и модулей
- Layer boundaries — что кому можно импортировать
- Соглашениями по именованию
- Паттернами обработки ошибок

## Когда использовать

- В начале проекта (после `/aif`)
- После существенного рефакторинга
- Когда структура проекта стала неясной

## Важно

`ARCHITECTURE.md` читается всеми скиллами как контекст. Хорошая архитектурная документация улучшает качество планирования и реализации.
