# /aif-docs — Генерация документации

Создаёт и поддерживает документацию проекта. Генерирует README как landing page и детальные docs по темам.

## Использование

```
/aif-docs              # сгенерировать всю документацию
/aif-docs api          # обновить только API docs
/aif-docs readme       # обновить только README
```

## Что генерирует

- `README.md` — краткий landing page с описанием, установкой, быстрым стартом
- `docs/getting-started.md` — детальная инструкция по установке
- `docs/api.md` — документация API эндпоинтов
- `docs/architecture.md` — архитектурный обзор
- `docs/configuration.md` — переменные окружения и настройки

## Принципы

- README — краткий, ссылается на docs для деталей
- Docs — детальные, по одной теме на файл
- Синхронизирует с реальным кодом (читает routes, schema, env vars)
