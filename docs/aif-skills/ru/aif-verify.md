# /aif-verify — Проверка реализации

Проверяет что реализация полностью соответствует плану: все задачи выполнены, код компилируется, тесты проходят.

## Использование

```
/aif-verify
```

Запускай после `/aif-implement` перед коммитом.

## Что проверяет

1. Все задачи плана отмечены `[x]`
2. Нет пропущенных файлов из плана
3. Код компилируется / нет TypeScript ошибок
4. Тесты проходят
5. Соответствие ARCHITECTURE.md и RULES.md
6. Логирование добавлено там где нужно

## Результат

```
✅ Task 1: реализована
✅ Task 2: реализована
⚠️ Task 3: тест отсутствует
❌ Task 4: файл не найден
```

## После verify

- Если всё ок → `/aif-commit`
- Если есть замечания → `/aif-implement task N` для конкретной задачи
