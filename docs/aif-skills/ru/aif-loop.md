# /aif-loop — Запуск по расписанию

Запускает другой скилл или команду повторно по интервалу.

## Использование

```
/aif-loop 5m /aif-review        # запускать review каждые 5 минут
/aif-loop 1h /aif-security-checklist   # каждый час
/aif-loop 30s проверь статус деплоя    # каждые 30 секунд
```

## Формат интервала

| Суффикс | Значение                  |
| ------- | ------------------------- |
| `s`     | секунды                   |
| `m`     | минуты (по умолчанию 10m) |
| `h`     | часы                      |

## Когда использовать

- Мониторинг статуса деплоя
- Периодическая проверка health check
- Повторный запуск тестов пока не пройдут
- Polling внешнего API

## Остановка

Нажми Ctrl+C или закрой сессию Claude Code.
