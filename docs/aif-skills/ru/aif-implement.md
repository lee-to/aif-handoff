# /aif-implement — Реализация плана

Выполняет задачи из текущего плана (`PLAN.md`). Отслеживает прогресс, можно прерывать и продолжать.

## Использование

```
/aif-implement          # выполнить все задачи
/aif-implement task 3   # выполнить конкретную задачу
```

## Что делает

1. Загружает план и skill-context
2. Находит первую невыполненную задачу (`[ ]`)
3. Реализует задачу с логированием
4. Обновляет чекбокс в плане (`[x]`)
5. Переходит к следующей задаче
6. По завершении — предлагает запустить `/aif-verify`

## Прогресс

Прогресс сохраняется прямо в `PLAN.md` через чекбоксы. Если сессия прервалась — просто запусти снова, продолжит с места остановки.

## Правила

- Не создаёт коммиты (это задача `/aif-commit` или approve в Handoff UI)
- Добавляет логирование к каждому значимому изменению
- Следует правилам из ARCHITECTURE.md и RULES.md

## После реализации

```
/aif-verify   # проверить что всё реализовано по плану
/aif-commit   # создать коммит
```
