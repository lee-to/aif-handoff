# /aif-fix — Исправление бага

Исправляет конкретный баг. Два режима: немедленное исправление или сначала создаёт план (`FIX_PLAN.md`).

## Использование

```
/aif-fix                              # выполнить существующий FIX_PLAN.md
/aif-fix кнопка submit не работает   # исправить конкретный баг
/aif-fix plan утечка памяти в WebSocket  # сначала создать план, потом исправить
```

## Режимы

| Режим             | Описание                              |
| ----------------- | ------------------------------------- |
| Без аргументов    | Выполняет существующий `FIX_PLAN.md`  |
| С описанием бага  | Немедленно исправляет                 |
| `plan` + описание | Создаёт FIX_PLAN.md, потом исправляет |

## Что делает

1. Читает skill-context для проектных правил
2. Исследует кодовую базу вокруг проблемы
3. Исправляет баг с добавлением логирования
4. Предлагает тесты для покрытия
5. Создаёт patch-файл для `/aif-evolve` (самообучение)

## Patch-файл

После каждого исправления создаётся `.ai-factory/patches/YYYY-MM-DD.md` с описанием проблемы и prevention points. `/aif-evolve` использует эти патчи чтобы улучшить скиллы и не повторять ошибки.

## После исправления

```
/aif-evolve aif-fix   # обновить скилл на основе нового патча
/aif-commit           # создать коммит
```
