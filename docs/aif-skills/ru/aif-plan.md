# /aif-plan — Планирование фичи

Создаёт детальный план реализации с разбивкой на задачи. Два режима: быстрый (fast) и полный (full).

## Использование

```
/aif-plan добавить экспорт в CSV
/aif-plan fast новый дашборд
/aif-plan full рефакторинг auth модуля
```

## Режимы

| Режим  | Когда использовать                              |
| ------ | ----------------------------------------------- |
| `fast` | Небольшие фичи, hotfix, понятные задачи         |
| `full` | Крупные фичи, рефакторинг, нужна ветка/worktree |

По умолчанию — `fast`.

## Что делает

1. Читает контекст проекта (DESCRIPTION, ARCHITECTURE, RULES, skill-context)
2. Исследует кодовую базу параллельными agentами
3. Создаёт план с задачами, зависимостями, фазами
4. Опционально создаёт git ветку и worktree (в режиме `full`)

## Результат

Файл `.ai-factory/PLAN.md` с задачами вида:

```markdown
- [ ] Task 1: Название задачи
  - Файл: path/to/file.ts
  - Зависит от: —
```

## После планирования

```
/aif-improve   # если хочешь доработать план
/aif-implement # когда план готов — начать реализацию
```
